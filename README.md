# 🎵 Jane Doe - Website

Projeto de Front-End desenvolvido para praticar a criação de um **site multipágina utilizando HTML e CSS**.

O site apresenta a página de uma artista fictícia chamada **Jane Doe**, contendo seções de apresentação, músicas e contato.

---

## 🚀 Páginas do site

- **Home** – Apresentação da artista e conceito do álbum  
- **About** – Informações e comentários sobre a artista  
- **Music** – Lista de músicas e vídeos incorporados  
- **Contact** – Formulário de contato, mapa e informações de gerenciamento

---

## 🛠 Tecnologias utilizadas

- HTML5  
- CSS3  

---

---

## 🎯 Objetivo

Praticar conceitos básicos de **desenvolvimento Front-End**, como:

- Estruturação de páginas com HTML
- Navegação entre páginas
- Organização de layout com CSS
- Formulários e elementos de mídia (vídeo e mapa)

---

## ▶️ Como executar

Clone o repositório e abra o arquivo `index.html` no navegador.
